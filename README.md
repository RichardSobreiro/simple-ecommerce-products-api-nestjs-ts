# REST API using Nest.js and Typescript

An example REST API using Nest.js and Typescript.

## Creating a Nest.js Project

- Install Nest.js Command Line Interface

```
npm i -g @nestjs/cli
```

- Install Nest.js Command Line Interface

```
nest new productsapi
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
